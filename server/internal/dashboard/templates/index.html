OcTool — 8 Arms

OcTool monitors your Copilot CLI sessions and fires one of 8 arms when it detects inefficiency:

ArmTriggers when…Threshold
redundant-readSame file read multiple times≥ 3 reads of same file
build-loopRepeated build/test failures≥ 3 build cycles
stale-contextInjected context is outdatedstaleness_risk = high
token-wasteHigh estimated waste tokens≥ 2000 waste tokens
low-edit-ratioToo many views vs editsview:edit ratio > 10
still-followupUnresolved "still" prompts≥ 2 still followups
vague-promptPrompt lacks specificityprompt_high ≥ 2
context-driftSession drifting from goalsession entropy spike

The dashboard auto-refreshes the status bar every 30 seconds.
